Encoding method: XOR cipher + Base64 envelope. The JSON payload is XOR'd byte-by-byte against a cyclic key string, then Base64-encoded into a wrapper object. The device firmware holds the same key and reverses the process on boot. This is obfuscation — not full cryptographic encryption — but it prevents casual inspection and protects the frequency/segment structure from plain-text discovery. For stronger protection, replace the XOR step with AES-128 in firmware; the JSON envelope structure stays identical.
